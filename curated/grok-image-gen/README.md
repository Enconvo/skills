# Grok Image & Video Generator

Generate images and videos using Grok AI via Chrome browser automation.
通过 Chrome 浏览器自动化调用 Grok AI 生成图片和视频。

## Features / 功能

- **Image Generation** — Text-to-image via Grok on x.com (often returns 4 variations) / 通过 x.com 上的 Grok 文字生成图片（通常返回4张变体）
- **Video Generation** — Text-to-video via grok.com/imagine with aspect ratio control / 通过 grok.com/imagine 生成视频，支持宽高比设置
- **Reference Images** — Upload reference images for style transfer, variations, and edits / 上传参考图用于风格迁移、变体生成和编辑
- **Image-to-Video** — Animate reference images into video clips / 将参考图动画化为视频片段
- **Session Reuse** — Leverages existing Chrome login session, no separate auth needed / 复用 Chrome 登录会话，无需额外认证
