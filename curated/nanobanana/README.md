# Nano Banana

AI image generation via Google Gemini with native aspect ratio and resolution control.
基于 Google Gemini 的 AI 图片生成，原生支持宽高比和分辨率控制。

## Features / 功能

- **Native AR Control** — 10 aspect ratios (16:9, 9:16, 1:1, 3:2, 4:5, 21:9, etc.) with exact pixel output / 10种宽高比原生支持，输出精确尺寸
- **Resolution Tiers** — 1K (drafts), 2K (default, screen use), 4K (print quality) / 三档分辨率：1K草稿、2K屏幕、4K印刷
- **Two Models** — Pro (higher quality, ~16s) and Flash (3.4x cheaper, ~8s) / 双模型：Pro高质量 / Flash高性价比
- **Reference Images** — Style transfer and variations via vision input / 支持参考图进行风格迁移和变体生成
- **Design Integration** — Built-in AR/size presets for PPTX, DOCX, and XLSX workflows / 内置适配演示文稿、文档等设计场景的预设
- **Dimension Verification** — Auto-checks output matches requested aspect ratio / 自动校验输出尺寸是否匹配请求
