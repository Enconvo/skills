# Image Prompt Enhancer

Add real camera specs and anti-AI directives to any image prompt for photorealistic results.
为图片提示词添加真实相机参数和反AI指令，生成更逼真的照片效果。

## Features / 功能

- **Camera Specs** — Auto-selects real camera body, lens, and settings (Sony, Canon, Fujifilm, Leica, iPhone) / 自动匹配真实相机机身、镜头和参数
- **Anti-AI Directives** — Fights plastic skin, fake bokeh, HDR artifacts, and AI smoothing / 消除塑料感皮肤、假散景、HDR伪影等AI痕迹
- **Camera Auto-Detection** — Detects selfie/film/DSLR intent from prompt keywords / 根据提示词关键词自动识别拍摄风格
- **Realism Levels** — Three tiers: standard (3 directives), high (7), ultra (10) / 三档真实度：标准、高、超高
- **Non-Invasive** — Only adds camera tech layer; never overrides lighting, mood, or creative intent / 仅叠加技术层，不干预创意意图
- **Pipeline Ready** — Flat text output mode for piping directly into Nano Banana / 支持纯文本输出，可直接传入图片生成工具
