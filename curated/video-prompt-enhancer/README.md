# Video Prompt Enhancer

Add real cinematography specs, camera movement, and anti-AI directives to any video prompt.
为视频提示词添加真实摄影参数、运镜指令和反AI指令，提升生成视频的真实感。

## Features / 功能

- **Cinema Camera Specs** — Auto-selects real cine camera and lens (Arri, RED, Sony Venice, Blackmagic) / 自动匹配专业电影机和镜头参数
- **Camera Movement** — Single clear movement instruction: static, push, pull, orbit, tracking, or handheld / 单一明确的运镜指令，避免多运镜混乱
- **5 Styles** — Cinematic, documentary, handheld, smartphone, and VHS/found-footage / 五种风格：电影、纪录片、手持、手机、VHS复古
- **Anti-AI Directives** — Fights subtitles, fade transitions, smoothing, and plastic textures / 消除字幕、转场、平滑和塑料质感等AI痕迹
- **Realism Levels** — Three tiers: standard (3 directives), high (8), ultra (13) / 三档真实度：标准、高、超高
- **Non-Invasive** — Only adds camera tech layer; never overrides dialogue, audio, or creative intent / 仅叠加技术层，不干预对白、音频或创意意图
- **Multi-Generator** — Works with Veo 3, Seedance, Grok, Kling, Runway, and more / 兼容多种视频生成器
